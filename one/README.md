## 个人项目

### 环境说明

* python 3.6+
* Window
* 项目中文编码为 utf-8，即使用终端运行时，需要设置终端的编码为utf-8,不然会乱码

### 文件说明

* File.py  将题目写入文件，即处理文件夹、文件的操作
* Practice.py   负责随机生成题目
* User.py   负责身份验证
* Main.py   客户端入口

### 运行

1. 在Main.py 目录下运行 `python Main.py`

2. 然后根据程序提示执行操作
